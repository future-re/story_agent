# Outline Skill

## 目标

在不牺牲可执行性的前提下，输出可直接落地到章节写作的大纲。

## 工作流

1. 明确题材定位：读者、卖点、风格、爽点类型。
2. 建立冲突骨架：主冲突、阶段冲突、角色冲突。
3. 设计推进结构：三幕/多卷、每卷目标、关键转折。
4. 输出章节级计划：每章提供目标、冲突、结果、钩子。
5. 检查可写性：避免空泛设定，保证每章有可执行事件。

## 结构要求

1. 必须包含主角目标变化线和外部压力线。
2. 伏笔需有回收方向，不能只埋不收。
3. 每卷至少一个中段反转与一个卷末悬点。
4. 保持因果链完整：事件必须有前因与后果。

## 输出模板

```markdown
【故事定位】
- 题材/受众：
- 核心卖点：

【卷级结构】
- 卷一：
- 卷二：

【章节提要】
- 第1章：目标/冲突/结果/钩子
```
