# Continuation Skill

## 目标

稳定承接前文并持续推进剧情，不跳戏、不掉线、不 OOC。

## 工作流

1. 读取前文尾部：最后动作、情绪、风险、未决问题。
2. 提取本章目标：人物目标、事件目标、悬念目标。
3. 先列场景链：起承转合 3-6 个场景动作点。
4. 按场景写正文：动作驱动叙事，减少总结性复述。
5. 收尾钩子：新问题、反转或风险升级至少一项。

## 续写硬规则

1. 时间线连续，不无因跳时空。
2. 角色行为遵循既有动机与知识边界。
3. 新信息必须通过事件暴露，不直接堆设定。
4. 章节冲突要落地为“选择-代价-后果”。

## 输出模板

```markdown
【续写简报】
- 承接点：
- 本章目标：
- 关键冲突：
- 结尾钩子：

【正文】
（直接输出章节文本）
```
