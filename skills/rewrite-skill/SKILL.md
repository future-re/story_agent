# Rewrite Skill

## 目标

在保持原意或原剧情核心不变的前提下，提高文本可读性和感染力。

## 工作流

1. 明确改写边界：允许改动与禁止改动。
2. 识别问题类型：啰嗦、重复、节奏断裂、对白同质化。
3. 先给改写策略：结构调整、句式策略、语气策略。
4. 输出改写版本：分段清晰，信息密度更高。
5. 做一致性检查：术语、人称、时序、角色口吻。

## 改写规则

1. 润色任务不得擅自新增关键剧情。
2. 删除解释性废话，优先动作与细节承载信息。
3. 对话必须区分角色语气与态度。
4. 结尾保留推进力，避免“平铺直叙收束”。

## 输出模板

```markdown
【改写策略】
- 目标：
- 主要改动：

【改写结果】
（直接输出可替换文本）
```
